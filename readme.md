# Development guide[Linux only]

## Clone the repo

```bash
git clone https://github.com/MakeMyBrain/auth-service
auth-service
npm run setupenv
```

## Generate .env

```bash
touch .env
```

### .env


## Start Server[development]

```bash
npm run dev
```
